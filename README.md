# BSO Baseball System

野球のライブ配信やOBSでの利用を想定した、ブラウザベースのBSO（ボール・ストライク・アウト）カウント表示システムです。

## 主な機能

- **リアルタイム更新**: Socket.io を使用し、操作GUIでの変更をディスプレイ画面へ即座に反映。
- **グローバルホットキー**: Windowsのバックグラウンドでもキーボード入力を検知し、カウント更新が可能。
- **統合操作GUI**: チーム名、得点、イニング、ランナー状況を直感的に操作。
- **スコアボード連動**: イニングごとの得点履歴を自動記録・編集可能。
- **設定保存**: チーム名やキーバインド設定を `config.json` に保存し、再起動後も保持。

## セットアップ

### 1. インストール
プロジェクトルートにある `install.bat` を実行してください。必要な依存ライブラリ（Express, Socket.io, node-global-key-listener等）がインストールされます。

### 2. 起動
`start-bso-system.bat` を実行するとサーバーが起動します。

## 画面一覧

サーバー起動後、以下のURLにアクセスして使用します（デフォルトポート: 3000）。

- **操作GUI (Control)**: `http://localhost:3000/control.html`
    - カウント、チーム情報、キーバインドの設定を行います。
- **表示画面 (Display)**: `http://localhost:3000/display.html`
    - OBSのブラウザソースとして読み込む画面です（1920x1080想定）。
- **スコアボード (Scoreboard)**: `http://localhost:3000/scoreboard.html`
    - スコアボードのみを表示する画面です。
    - **URLパラメータ**: `?nobg=true` を付けると、背景画像（`table_bg.png` と `table_line.png`）を非表示にし、テキストのみを表示します。OBS側で自前の背景を使いたい場合に有効です。

## ホットキー操作

デフォルトでは以下のキーが割り当てられています（操作GUIから変更可能）。

- `B`: ボール追加
- `S`: ストライク追加
- `O`: アウト追加
- `1` / `2` / `3`: 1塁 / 2塁 / 3塁ランナー切替
- `I`: イニング増加
- `T`: 表裏切り替え
- `U`: 先攻得点追加
- `D`: 後攻得点追加
- `R`: カウントリセット

## ディレクトリ構成

- `public/`: 各画面のHTML/JSファイル
- `assets/`: カウントやランナーの画像リソース
- `server.js`: バックエンドサーバー（ホットキー検知・状態管理）
- `config.json`: 永続化される設定ファイル
- `PLAN.md`: 開発時の要件定義メモ
- `UPDATES.md`: 作業履歴
